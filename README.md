**有点搞懂了理清了一下大致步骤**
> 1. GitHub上先建立一个库（repository）
> 2. 运行`git clone git@github.com:D41E/AT-1ST.git`把库复制下来
> 3. 在本地进行编辑
> 4. 运行`git add .`缓存更新文件
> 5. 运行`git commit -m"****"`申明确认提交
> 6. 运行`git push origin master`上传
> 
> *几点注意：多用`git status`查看库状态，先从GitHub上PULL下库，再进行编辑操作，确保有序工作。*
